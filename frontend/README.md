## Quick start

1. Unzip and open this folder in terminal.
2. Install dependencies:
   ```
   npm install
   ```
3. Run dev server:
   ```
   npm run dev
   ```
4. Open the URL printed by Vite (usually http://localhost:5173).

This project includes:
- React + Vite
- TailwindCSS configured (files present)
- Pages: Login, Register, CustomerDashboard, OfficerDashboard, ApplyLoan, LoanStatus
- Components: Navbar, PrivateRoute
- Context: AuthContext
- API: axios instance

You may need to run `npm install` to fetch dependencies. Tailwind is included in devDependencies and the config files are present.
